# EdTech — Платформа для тестирования разработчиков

Интерактивная EdTech платформа на Django, созданная для проверки знаний по различным технологическим стекам (Python, DevOps, Frontend и др.). Проект включает систему геймификации, таблицу лидеров и детальную аналитику результатов.

## 🚀 Технологии

* **Backend:** Django 5.x
* **Database:** PostgreSQL
* **Frontend:** HTML5, CSS3 (Modern UI/UX), JavaScript
* **Infrastructure:** Docker & Docker Compose
